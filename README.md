# 图书馆管理系统 java955

## 项目概述

图书馆管理系统是一款基于前后端分离模式开发的应用，旨在提供便捷的图书管理及借阅服务。该项目采用Java和SpringBoot进行后端开发，Vue.js以及相关技术栈实现前端界面。

## 技术栈

### 后端技术

- SpringBoot
- MyBatis-Plus

### 前端技术

- Vue.js
- Axios
- Element-UI

### 开发环境

- Java 1.8
- Node.js
- MySQL 8
- Maven

## 功能模块

### 管理员端

- 首页
- 系统管理
  - 用户管理
  - 菜单管理
  - 角色管理
- 读者管理
  - 读者列表
- 图书管理
  - 图书分类
  - 图书列表
- 借阅管理
  - 借书管理
  - 还书管理
- 借阅查看
- 公告管理
  - 公告列表

### 读者端

- 借阅管理
- 借阅查看
- 公告管理
  - 公告列表

## 系统角色

- 管理员：负责图书馆的日常管理工作，如用户管理、图书管理、借阅管理等。
- 读者：可以查看图书信息、借阅图书、查询借阅记录及公告。

## 运行环境

- 后端：Java运行环境
- 前端：现代浏览器
- 数据库：MySQL 8

## 部署步骤

1. 准备Java 1.8及以上版本环境。
2. 安装并配置MySQL 8数据库。
3. 使用Maven构建后端项目。
4. 部署前端项目至Web服务器。
5. 运行后端服务。
6. 打开前端界面，进行访问。
